# 颜色选择器

颜色选择

## api

### props

| 参数               | 说明               | 类型   | 默认值      |
| ------------------ | ------------------ | ------ | ----------- |
| theme              | 主题 light 或 dark | string | light       |
| color              | 当前选中颜色       | string | #000000     |
| quickColorsDefault | 快捷颜色默认列表   | Array  | ['#000000'] |
